# Dog Profile Android Design

This Android application showcases a simple dog profile UI using Jetpack Compose.

## Overview

The app consists of a profile page displaying a dog's information, including an image, name, location, follower count, following count, and post count. Users can also follow the dog or send a direct message.



## Features

- Circular profile image with a red border.
- Name displayed with a green color and specific font style.
- Location information.
- Statistics section showing followers, following, and posts.
- Follow and Direct Message buttons.

## Installation

To run the app, you need Android Studio and a device or emulator with Android API level 21 or higher.

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/dog-profile-android.git
   ```

2. Open the project in Android Studio.

3. Run the app on your device or emulator.

## Usage

Feel free to use this project as a starting point for your own dog profile or customize it for different purposes. Explore and experiment with Jetpack Compose to enhance the UI further.

## Dependencies

- Android Jetpack Compose
- Kotlin
- androidx.compose.material3
- androidx.compose.foundation
- ...

## Preview

See a preview of the profile page by running the `Preview` composable function in Android Studio.

```kotlin
@Preview(showBackground = true)
@Composable
fun Preview(){
    ProfilePage()
}
